# todolistapp_vuex4_router4
* vuex 4.x + vue-router 4.x + vue 3.x 을 이용한 TodoList App 예제 샘플

### 실행 방법
~~~
cd todolistapp_vue3
yarn install
yarn serve --open
~~~
